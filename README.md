# Responsive Portfolio Website

This project was developed as part of my Full Stack Development Internship at DecodeLab.

A modern, responsive, and user-friendly portfolio website built using HTML, CSS, and JavaScript. The project demonstrates responsive web design principles, semantic HTML structure, CSS Grid & Flexbox layouts, and JavaScript interactivity.

---

## 🚀 Features

- Responsive Design (Mobile, Tablet, Desktop)
- Semantic HTML5 Structure
- CSS Flexbox Layout
- CSS Grid Layout
- Mobile Navigation Menu
- Dark Mode Toggle
- Contact Form Validation
- Smooth Scrolling
- Scroll Animations
- Accessible and User-Friendly Interface
- Clean and Modern UI Design
